The proceedings contain 23 papers. The topics discussed include: noisy phoneme recognition using 2D convolution neural network; the synthetic data application in the UAV recognition systems development; demonstration of 50 ps per-position differential pulse position modulation data transmission; supercapacitor balancing circuit design considerations for high current charge-discharge cycles; diagnostics of pre-fault conditions using the impact of electrolytic capacitor aging on power supply dynamics; the latest in natural language generation: trends, tools and applications in industry; brain-computer interfacing: developments, perspectives and limitations; analysis on RGB dataset requirements for remote inspection of power grid infrastructure; machine learning-based sensor data forecasting for precision evaluation of environmental sensing; and control unit for thermoelectrically configurable medical lighting equipment.
