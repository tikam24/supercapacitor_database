The proceedings contain 317 papers. The topics discussed include: design and application of distribution automation system in intelligent circulating power supply system management; automatic pole line fault detection system using microcontroller; fusion of resource sharing system development based on block knowledge management system and Hadoop; design of decentralized online guidance platform for college English translation based on blockchain networking algorithm; intelligent and secured medical system based on remote visual operation platform; real-time and secure online event voting management system; a survey and taxonomy of anti-phishing techniques for detecting fake websites; cloud based smart mobile application for women safety; and an improved carbon cloth structure for asymmetric supercapacitor design.
