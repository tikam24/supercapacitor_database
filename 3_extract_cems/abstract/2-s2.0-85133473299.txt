The proceedings contain 61 papers. The topics discussed include: a birdsong denoising using cascaded combination of phase spectrum compensation and wavelets; performance analysis of mmc with novel multi-carrier phase shifted pulse width modulation (MCPS-PWM); wide band conformal coplanar Benz shaped circular ring antenna for C and X band applications; developments of approximate computing: from algorithm level to system level; DFIG HVDC based wind farms system profile improvement under grid faults; performance analysis of OMP hybrid beamforming algorithm for OFDM millimeter wave MIMO systems; an efficient approach for sentiment analysis using data mining algorithms; battery and supercapacitor performance analysis during regenerative braking in electric vehicle; a comparative analysis on frequency control in an autonomous microgrid using adaptive optimized fuzzy controllers; and prediction of human suicidal tendency based on social media using recurrent neural networks through LSTM.
