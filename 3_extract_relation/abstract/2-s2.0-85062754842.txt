The proceedings contain 60 papers. The topics discussed include: number recognition of parts book schematics using convolutional recurrent neural network; heart rate modeling and SMC simulation during treadmill exercise; robust three-dimensional mapping with balancing system; color recognition method based on circle extraction and similarity; efficient super capacitors fuelled by smart thermoelectric cloths to charge smart devices; application of SAC extended using null-space to a rotation control system; eco-friendly billboard by using Li+batteries and supercapacitor parallel connection; large-scale microwave analysis based on domain decomposition method; and assessment of upper limb skill for pedaling exercise.
