The proceedings contain 146 papers. The topics discussed include: intelligent data acquisition system for quartz crystal microbalance and other resonator-based sensors; distributed real-time measurement system using time-triggered network approach; method of ADC's nonlinearity identification; self-evaluation configuration for remote data logging systems; supercapacitor data acquisition systems; object oriented concurrent fault diagnostic system for 8051 based microcontroller system; temperature measurement system based on thermocouple with controlled temperature field; an automated data acquisition setup for electro-chemical measurements; transducers with magnetic microsensors; intelligent system of temperature field ecological monitoring; multi-agent based distributed measurement system for noise monitoring; discrete-time PID controller design in programmable logical controllers; and a power benchmark experiment for battery-powered devices.
